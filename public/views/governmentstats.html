About

Once the project is finished loading you will find a few graphs that i have written using the d3.js library. The data comes from an api which i developed using data from government apis. Please note that the data might not be completely accurate and does not represent all of the United States this data is only for the state of California. **Note this project is still work in progress. - more data will be added soon
